Styles

You can load CSS files if you need for particular modules. Since you are loading this css file with the module, it implies that it is not global styles, but rather styles for the particular module.

The css file is loaded in, then a unique id is generated for the css file based off of some of its file characteristics. This unique id is then prepended to each rule, and then the containing element gets the class with the id. The css is then inserted into the head with a style tag. This helps to scope the styles to the particular module instead of globablly.
